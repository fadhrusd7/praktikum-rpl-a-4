## Standar Commit Message

Kami menyepakati format **Bahasa Inggris** untuk pesan commit agar history Git tetap rapi dan mudah dipahami.

| Format                        | Kegunaan                            |
| ----------------------------- | ----------------------------------- |
| `Add [feature name]`          | Menambahkan fitur baru              |
| `Fix [bug description]`       | Memperbaiki bug atau kesalahan      |
| `Update [file/document name]` | Memperbarui konten atau dokumentasi |

## Mekanisme Eskalasi

1. Jika terjadi konflik internal atau hambatan teknis yang tidak dapat diselesaikan secara mandiri, anggota wajib segera melaporkan kepada **Ketua Tim**.
2. Jika masalah tidak kunjung terselesaikan dalam **24 jam**, Ketua Tim akan menghubungi **Asisten Praktikum** untuk meminta bantuan lebih lanjut.
